    # Goal Tracker Dashboard 
         ## Features - CRUD goals 
         - RTL/LTR 
        - Responsive 
        - XP system 
        - Streak tracking 
    
    ## Tech Stack 
    - React
     - Vite 
     - MUI 
     - React Router 
     - i18next 
     
     ## Streak Rules
      - Consecutive days increase streak
       - Missing a day resets streak
       
     ## XP Rules 
     - +20 XP per progress log 
     - +100 XP on completion 
     ## Run Project npm install npm run dev
